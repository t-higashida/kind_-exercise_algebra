\chapter{複素整数}

\section{SUMMARY}

(1) 有理整数$a,b$により，
\[
  \alpha = a+ ib , \: i = \sqrt{-1}
\]
と表させる複素数を\textbf{複素整数}または\textbf{Gauss 整数}という．

(2) 複素整数は複素平面上の\textbf{格子点}として図示される．

(3) 複素整数全体の集合$\Z[i]$は，複素数の三則（加・減・乗）に関して
閉じており，\textbf{Gauss 整数環}と呼ばれる．

(4) 複素整数$\alpha = a + ib$に対し，非負整数
\[
	N(\alpha) = |\alpha|^2 ( = \alpha \overline{\alpha} = a^2 + b^2)
\]
を$\alpha$の\textbf{ノルム}（norm）という．$N(\alpha) = 0 \Leftrightarrow \alpha = 0$．

例えば，$a=3+4i$のとき，$\overline{a}=3-4i$，$N(3+4i)=a\overline{a}	=	3^2	+	4^2	=25$．


(5) 正則な複素整数$\pm 1$，$\pm i$を\textbf{単数}という．これらは乗法に関して“群”を成す．

（中略）

(8) 単数，$0$以外の複素整数$\gamma$が自明な約数（同伴数と単数）の他には約数を持たないとき，
$\gamma$を\textbf{Gauss 素数}という．

(9) 単数，$0$以外の任意の複素整数$\alpha$は幾つかの素因数の積に，素因数の順序および
単数因数の違いを度外視して，一意的に分解される（素因数分解定理）．

(10) \textbf{Fermat - Euler の素数定理}

奇素数$p$（有理素数）について，
\[
	p \equiv 1 \mod 4 \Leftrightarrow \text{$p$は Gauss 素数ではない．}
\]
このとき，$p = \gamma \overline{\gamma}$（$\gamma,\overline{\gamma}$は共役な Gauss 素数）である．

3. 環準同型写像$f: R \rightarrow R'$について，次の事柄を証明せよ．
\begin{list}{}{}
\item	(1) $\text{Im } f$は$R'$の部分環である．
\item	(2) $\Ker f$は$R$のイデアルである．
\end{list}
\注意{}
環$R$の部分環の間の包含関係は Hasse の図式で図示される
（§15を参照）．これを利用して，環準同型写像：
$R \rightarrow R'$における$\Ker f$と$\text{Im }$の対応関係を
図示すれば，右図のようになる．

\証明

(1) $\text{Im } f$の任意の2元$a' =f(a),b' =f(b)$に対し，
\begin{align*}
	&a' - b' = f(a) -f(b) = f(a-b), \: a'b' = f(a)f(b)=f(ab). \\
  &\therefore a'-b', a'b' \in \text{Im } f.
\end{align*}
したがって$\text{Im } f$は$R'$の部分環である．

(2) $\Ker f$の任意の2元$a,b$と$R$の任意の元$x$に対し，
\begin{align*}
	&f(a-b) = f(a) -f(b) = 0' - 0' = 0', \\
  &f(xa)=f(x)f(a)=f(x)0' = 0', \: f(ax) = f(a)f(x) = 0' f(x) = 0'.\\
  &\therefore a-b, xa,ax  \in \Ker f.
\end{align*}
したがって$\Ker f$は$R$のイデアルである．

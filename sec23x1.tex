\chapter{環の準同型定理}

\subsection{SUMMARY}

(1) 環$R$のイデアルを$A$とする．加法群としての$R$の部分群$A$による
剰余群
\[
  R/A = \{A + a \: | \: a \in R\}
\]
は，
\begin{list}{}{}
\item	加法：$(A + a ) + (A + b) = A + a+b$, 
\item 乗法：$(A+a)(A+b) = A + ab$
\end{list}
のもとで環をなす．この環$R/A$を$R$の$A$に関する\textbf{剰余環}といい，
特に，$R/A$が体をなすときは，\textbf{剰余体}という．

(4) 環$R$から$R'$への写像$f$は，もしそれが条件
\[
  f(a+b)=f(a) + f(b) , \: f(ab)=f(a)f(b)
\]
を満たすならば，\textbf{(環)準同型写像}と呼ばれる．

（保留）

(6) 環$R$から環$R'$への準同型写像$f$において，
\begin{list}{}{}
\item	(a) $R$の$f$による\textbf{像（準同型像})
\[
  \text{Im } f = \{\: x' \in R' \: | \: x' = f(x) , x\in R\}
\]
は$R'$の部分環をなす．$f$が全射$\Leftrightarrow \text{Im } f = R'$.
\item (b) $R'$零元$0'$の原像を$f$の\textbf{核}という．核
\[
	\Ker f = \{ \: x \in G \: | \: f(x)=0' \: \}
\]
は$R$のイデアルをなす．$f$が単射$\Leftrightarrow \Ker f = \{0\}$.
\end{list}
